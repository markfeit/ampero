# Fryin' Seltzer

A sound inspired by Brian Setzer's usual rig.

## Controls

| Control | Description |
| ------- | ----------- |
| **Knob 1** | Distortion Gain |
| **Knob 2** | Delay Feedback |
| **Knob 3** | Reverb Mix |
| **CTRL** | Clean (Green) / Lightly Distorted (Red) |
| **EXP1** | Volume |
| **EXP2** | Increases gain on the drive pedal from 33 to 100. |
