Cross the bridge pickup, overdrive down to the beach and catch a wave or two.


## Controls

| Control | Description |
| ------- | ----------- |
| **CTRL** | Engaged (red) adds vibrato. |
| **EXP1** | Vibrato depth. |
| **EXP2** | Not Set |
| **Knob 1** | Tremolo Shape |
| **Knob 2** | Amp Presence |
| **Knob 3** | Reverb Decay |
